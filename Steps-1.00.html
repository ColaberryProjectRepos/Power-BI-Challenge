How to list Duplicate records in Power BI 1

Preparation:

  • Download the zipped .pbix Code File named PowerBI_Skills_Challenge_Q1.
  • Unzip and open the .pbix file in Power BI Desktop (the DAX Measures have already been created).
  • The canvas will be empty, but follow the steps and screenshot.

How to list Duplicate records in Power BI:

  1. Make sure the FIELD icon is highlighted
  2. Drag & Drop the field “name” from the "users" table, to an empty canvas, a table chart will appear on the canvas
  3. Drag and Drop the Measure named “Count All Usernames” to the table on the canvas
  4. From “FILTERS” / “Visual level filters” section, select the drop-down list of the “Count All Usernames” filter and select “is greater than”. Enter a 1 in the text-box
  5. Click "Apply filter"

Note: You can click on the paint roller icon to format the chart as you desire. We will have a tutorial on chart formatting soon.
